# Scrape-A-Rama
SCRAPE-A-RAMA

https://mysterious-garden-64783.herokuapp.com/

Overview

SCRAPE-A-RAMA is a web application that utilizes Mongoose and Cheerio to scrape articles from ScreenRant , and allows users to save articles and to make notes.
Live Site
https://mysterious-garden-64783.herokuapp.com/
